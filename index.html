Third party signin using AWS cognito

watch tutorial video

Steps

1. create a user pool
2. create an app client
3. create a domain
4. set up a google project

a. head to google developer console and create a project

b. set up your OAuth consent screen

c. create an OAuth credential and get your client id as well as client secret

5. link aws with google

a. create a new identify provider and paste your google client id/secret

b. head to your app client, edit hosted ui and add google to your identity provider
